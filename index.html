碩士研究Graduate Research

WeatherSAM — 惡劣天氣語意分割WeatherSAM — Adverse-Weather Segmentation

以 Meta Segment Anything Model（ViT-H）為骨幹的參數高效語意分割框架，針對濃霧等惡劣天氣下的 自動駕駛場景。結合 CLIP 文字提示、GeoCLIP 地理位置編碼與參考幀跨視角注意力融合， 並提出 Active Boundary Loss 強化邊界品質，僅需訓練約 10–20% 的參數。

A parameter-efficient semantic segmentation framework built on Meta's Segment Anything Model (ViT-H) for autonomous driving in adverse weather such as dense fog. It fuses CLIP text prompts, GeoCLIP location encoding, and cross-view attention over reference frames, and introduces an Active Boundary Loss for sharper boundaries — training only ~10–20% of parameters.

A multi-model perception system combining lane detection (CLRNet), drivable-area segmentation (YOLACT-Edge), traffic-light detection (YOLOv5), and OpenStreetMap map-matching to refine GPS positioning to lane-level accuracy. All models run in real time with TensorRT acceleration on an RTX 3080 Ti.
